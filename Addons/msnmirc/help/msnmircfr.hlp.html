Introduction

Voici la seconde version de mon fameux client MSN Messenger pour mIRC. Cette fois-ci, la mise à jour ne consiste pas qu'en quelques bug fixes, mais plutôt en un recoding intégral de l'addon! Cette version supporte maintenant les emoticons, les conversations multi-users, la réception de fichier (en développement), support multilingue, la création de ses thèmes de discussion avec tags, et plusieurs autres fonctions dont la traditionelle connexion en statut Apparaitre hors ligne. Cet addon utilise le protocol MSNP9, celui utilisé par MSN Messenger 6.0, l'ancienne version de l'addon utilisait MSNP7, une version maintenant désuète qui ne fonctionne plus. J'aimerais remercié plusieurs personnes, ce que je ferai dans les crédits tout en bas de cette page. Pour toute question, merci de me contacter par email.

Configurations

Onglet Général

› Vous pouvez vous connecter en statut Apparaitre hors ligne, ce que l'on ne peut pas faire avec MSN Messenger, donc vos contacts ne voient pas que vous vous êtes connectés.
› Lorsque vous vous connectez, vos contacts sont ajoutés dans des groupes qui sont fermés, pour voir les contacts lors de leur ajout, vous pouvez activer l'option pour ouvrir les groupes de contacts.
› Cette version de MSNMIRC support les sons wav, donc vous pouvez les activer pour lorsque vous recevez un message ou lorsque un contact se connecte.
› La fenêtre qui vous indique qu'un de vos contacts s'est connecté peut avoir deux styles, soit un design comme Windows, avec bordure 3D, ou bien une simple ligne noire comme contour, vous pouvez aussi choisir combien de temps vous voulez la garder ouverte avant qu'elle ne se ferme automatiquement.
› Pour l'ouverture d'un popup, vous pouvez choisir si vous voulez que le popup s'ouvre lors d'un click droit ou bien d'un clic gauche de la souris.
› Finalement, le support multilingue! Vous pouvez changer la langue de l'addon quand vous le voudrez.

Onglet Conversations

› L'option de mettre en gras le message que l'on envoie, cela s'affichera en gras sur l'écran de votre contact.
› Entre chaque message reçu, le choix de sauter une ligne ou non pour espacer la conversation.
› Chaque conversation est stockée sous un fichier HTML, le choix de garder ces fichiers pour les consulter après utilisation ou bien de les effacer.
› Lors de l'ouverture d'une conversation, vous pouvez réduire automatiquement cette fenêtre dans la barre des tâches, ce qui peut vous permettre de ne pas encombrer votre écran de fenêtres ouvertes lorsque vous êtes absents et que MSNMIRC est en fonction.
› Lorsque vous êtes absents sur IRC, MSNMIRC peut prendre le relais en annonçant à vos contacts que vous êtes absents avec le même message que celui sur IRC.
› Encore dans la configuration de votre thème de conversation, vous pouvez choisir entre l'affichage du pseudo ou de l'email de vos contacts.
› Si vous cochez la case pour créer votre propre thème de conversation, vous pourrez, à l'aide du HTML et de tags pré-formattés, obtenir un thème réellement à votre goût. Pour la liste des tags, consultez la section Tags juste ci-dessous.

Tags

Pour afficher l'heure: <time>
Pour afficher le pseudo ou l'email de la personne: <nick>

Vous pouvez également utilisez n'importe quel tag HTML comme <b>, <u>, <br>, <font>, etc..

Connexion

Pour se connecter, j'ai créé un raccourci, vous n'avez qu'à appuyer sur la touche F4, sinon il y a les popups. Vous n'avez qu'à mettre votre adresse email complète avec votre mot de passe pour vous connecter à MSN Messenger. Le dialog principal devrait s'ouvrir automatiquement par la suite. Si la connexion s'avère très lente, c'est souvent dû à Passport.NET avec qui je rencontre beaucoup de problèmes de lenteur et souvent le serveur est mort, donc je ne peux pas contourner le problème, c'est semblable à MSN Messenger et je dois utiliser ces serveurs.

Dialog principal

Vos contacts seront situés dans les différents groupes, double-click sur le groupe pour l'ouvrir ou le fermer. Par la suite, utilisez les popups pour gérer vos contacts, dialoguer avec lui, ou voir son profil. Les menus situés dans le haut du dialog servent à avoir un accès rapide à quelques commandes utiles comme les configurations, l'aide ou l'ajout d'un contact.

Conversation

Pour envoyer un message, écrire le message dans l'edit box dans le bas, et appuyer sur Enter pour l'envoyer. Pour mettre un emoticon dans son message, cliquer sur le bouton à droite de Envoyer, puis choisir l'emoticon que vous voulez inclure dans le message. L'onglet Liste Des Usagers vous donnera la liste des usagers qui participent actuellement à la conversation, donc qui recevront vos messages. Pour inviter un contact, aller dans l'onglet du même nom et choisissez un contact, puis cliquez sur Inviter, dans la conversation, s'il a bien été invité, vous le verrez vous rejoindre. Pour imprimer la conversation en cours, il suffit de cliquer sur le bouton Imprimer.

Réception de fichiers

Ceci n'est pas terminé, donc vous pouvez rencontrer plusieurs problèmes lors de la réception de fichiers.

Lorsqu'un de vos contacts vous envoie un fichier, vous recevrez un avis, et vous pourrez soit accepter, soit refuser. Si vous acceptez, vous ne verrez pas le progrès de téléchargement du fichier, donc vous pourrez demander à la personne qui vous l'envoie. Par contre, lorsque le téléchargement sera terminé, vous serez notifiez de la fin du téléchargement.
